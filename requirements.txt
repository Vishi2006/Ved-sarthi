# HealthMeet - Project Requirements

This is a Node.js project split into a frontend (`client`) and a backend (`server`). 
To run this code, you do NOT need a Python `requirements.txt`. Instead, you will use `npm install` to install the Node packages from the `package.json` files.

## System Requirements
- Node.js (v18.x or higher recommended)
- npm (comes with Node.js)

---

## 1. Frontend Requirements (`client` folder)
The frontend is built with React 19, Vite, and Tailwind CSS.
Change directory to the `client` folder and run `npm install` to get these packages:

### Dependencies:
- react (^19.2.0)
- react-dom (^19.2.0)
- react-router-dom (^7.13.1)
- tailwindcss (^4.2.1)
- @tailwindcss/vite (^4.2.1)

### Dev Dependencies:
- vite (^7.3.1)
- @vitejs/plugin-react (^5.1.1)
- eslint (^9.39.1)

### To Run:
cd client
npm install
npm run dev

---

## 2. Backend Requirements (`server` folder)
The backend is an Express server connecting to MongoDB using Mongoose.
Change directory to the `server` folder and run `npm install` to get these packages:

### Dependencies:
- express (^5.2.1)
- mongoose (^9.2.4)
- cors (^2.8.6)
- dotenv (^17.3.1)
- bcryptjs (^3.0.3)
- jsonwebtoken (^9.0.3)

### To Run:
cd server
npm install
npm start (or node server.js)
